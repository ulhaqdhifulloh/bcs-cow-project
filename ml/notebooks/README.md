# Notebooks

Direktori ini menyimpan notebook Jupyter untuk eksplorasi dan pipeline utama ML.

Untuk fase PoC, seluruh tahapan pipeline (ringkasan business understanding singkat → EDA → data preparation → feature extraction → training → evaluasi → simpan/download model) ditempatkan di **satu notebook utama**:

- `01_bcs_full_pipeline.ipynb` – notebook utama pipeline ML.

Notebook tambahan (mis. untuk eksperimen khusus) boleh ditambahkan kemudian jika diperlukan, tetapi implementasi baseline mengikuti satu pipeline notebook ini sesuai `tech-specs/ml_tech_specs_v1.1.md`.
